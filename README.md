# AOD Storyteller Archetypes Quiz

This project is a React application built with Vite, TypeScript, and Tailwind CSS.
It features an interactive quiz to help users discover their storytelling archetype in the context of AOD (Alcohol and Other Drugs) harm reduction work.

## Features

- Interactive quiz with scoring logic.
- Visual results with primary and secondary archetypes.
- Built with React and Lucide React icons.
- Styled with Tailwind CSS.

## Getting Started

1. Install dependencies:

   ```bash
   npm install
   ```

2. Run the development server:

   ```bash
   npm run dev
   ```

3. Build for production:

   ```bash
   npm run build
   ```
